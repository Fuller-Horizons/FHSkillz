# JAIL-WAYFIND

Fog navigation for work too big for one session — name the destination, chart a map of DECISION tickets (questions, not build slices) with blocking edges, resolve one at a time via the kernel until the way is clear, then hand off to execution.

Part of the **[FHSkillz](../../README.md)** collection — JAIL kernel layer. Version 1.0.0. Adapted from Matt Pocock's wayfinder (MIT), generalized past issue trackers. See [`SKILL.md`](SKILL.md); routing in [`docs/skill-graph.md`](../../docs/skill-graph.md).
