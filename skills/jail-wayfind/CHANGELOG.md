# Changelog — jail-wayfind

## 1.0.0
- New kernel skill. Fog navigation for work too big for one session — name the destination, chart a map of DECISION tickets (questions, not build slices) with blocking edges, resolve one at a time via the kernel until the way is clear, then hand off to execution. Adapted from Matt Pocock's wayfinder (MIT), generalized past issue trackers.
