# JAIL-PY-PROMPT-TOOLS

The runnable half of [`jail-prompt`](../jail-prompt/): five stdlib-only Python checks (secret scan, prompt lint, chain lint, truth lint, dry-run) that back jail-prompt's verification steps with real exit codes. jail-prompt works without this skill (manual fallbacks); install both for the full loop.

Part of the **[FHSkillz](../../README.md)** collection. Version 1.0.0. Requires code execution; Python 3.8+, no third-party deps. See [`SKILL.md`](SKILL.md) for the script table and usage.
