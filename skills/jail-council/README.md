# JAIL-COUNCIL

Maximum-accuracy deliberation: 3–5 independent models answer blind, adversarially review each other's **anonymized** answers, disputed facts get a targeted verification round, and a chairman synthesizes **by evidence, never by vote** — dissent preserved, audit appendix attached. Cost is disclosed, never a gate: invoking the council means accuracy outranks tokens.

Part of the **[FHSkillz](../../README.md)** collection — JAIL kernel layer. Version 1.0.0. Pattern inspired by Andrej Karpathy's [llm-council](https://github.com/karpathy/llm-council) (no code reused). Cross-provider (Tier A) councils are native in **OpenCode CLI** — verified per-agent model config in [`references/opencode-runbook.md`](references/opencode-runbook.md); protocol texts and output templates in [`references/council-protocol.md`](references/council-protocol.md).
