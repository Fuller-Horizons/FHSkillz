# JAIL-SKILL-MINER

Mines codebases and histories for plugin-worthy disciplines, filters features out, dedupes against installed skills, and stops for approval before authoring.

Part of the **[FHSkillz](../../README.md)** collection — JAIL kernel layer. Version 1.0.0. See [`SKILL.md`](SKILL.md) for the full discipline; routing and handoffs in [`docs/skill-graph.md`](../../docs/skill-graph.md); system rules in [`docs/JAIL-CONSTITUTION.md`](../../docs/JAIL-CONSTITUTION.md).
