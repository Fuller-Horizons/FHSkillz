# JAIL-PY-RATE-TOOLS

The runnable half of [`rate-skill`](../rate-skill/): four stdlib-only Python tools — validate a rating record, save it to history with delta reporting, measure score variance across repeated runs, and structurally lint a skill directory. rate-skill works without this skill (manual fallbacks); install both for the full loop.

Part of the **[FHSkillz](../../README.md)** collection. Version 1.0.0. Requires code execution; Python 3.8+, no third-party deps. See [`SKILL.md`](SKILL.md) for the script table and usage.
