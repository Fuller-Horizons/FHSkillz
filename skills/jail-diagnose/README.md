# JAIL-DIAGNOSE

Feedback-loop-first diagnosis of hard defects — build a tight red-capable reproduction signal before any hypothesizing, minimize, instrument where hypotheses predict divergence, fix the confirmed cause, and ship a mandatory regression case.

Part of the **[FHSkillz](../../README.md)** collection — JAIL kernel layer. Version 1.0.0. Adapted from Matt Pocock's diagnosing-bugs (MIT), generalized beyond code and bound to JAIL evidence rules. See [`SKILL.md`](SKILL.md); routing in [`docs/skill-graph.md`](../../docs/skill-graph.md).
